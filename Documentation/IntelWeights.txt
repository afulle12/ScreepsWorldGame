================================================================================
CHANGES FROM PREVIOUS VERSION
================================================================================

1. DUAL PURPOSE — Power spawn fueled (4%) split into two independent checks:
     - Power spawn has energy: 3% (energy store > 0 in power spawn)
     - Power in room:          1% (power resource > 0 in storage, terminal,
                                   OR power spawn — checked separately)

2. DUAL PURPOSE — Extensions now give proportional partial credit when below
   expected count (not binary). Full 10% only at or above expected count.

3. ECONOMIC — Missing source infrastructure: -12% applies when ANY source
   lacks a link or container within range 3 (not only if ALL sources lack it).

4. DUAL PURPOSE — Hauler check uses ALL creeps in room (FIND_CREEPS), not
   just owned creeps (FIND_MY_CREEPS). Useful for scouting enemy rooms.

5. ECONOMIC — Level 1-5 commodity check also includes Level 0 factory outputs:
   composite, crystal, liquid.

6. MILITARY — Labs active: a lab counts as active if cooldown > 0 OR it has
   a mineralType loaded with resources in its store (not cooldown alone).
